## 📌 Project Overview

The **Healthcare Power BI Dashboard** is an interactive Business Intelligence solution designed to help Hospital Management Teams monitor, analyze, and optimize healthcare operations through data-driven insights.

This dashboard provides a centralized view of key hospital metrics including patient visits, revenue analysis, appointment waiting times, bed availability, staff information, and patient feedback.

---

## 🎯 Project Objectives

✅ Create interactive and insightful healthcare visualizations

✅ Perform data cleaning and transformation using Power Query

✅ Design an efficient data model with relationships

✅ Develop meaningful KPIs for hospital performance tracking

✅ Implement drill-down and drill-through functionalities

✅ Enable advanced filtering and slicing for better analysis

✅ Enhance user experience with custom visuals and navigation
